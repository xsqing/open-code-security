# open-code-security
Find code security vulnerabilities automatically using LLM.Inspired by Claude Code Security.

## Introduction
This project leverages LLMs to detect code security issues, anti-patterns, common vulnerabilities, and hidden auxiliary vulnerabilities in source code. It provides an AI-powered alternative to traditional static application security testing (SAST) tools, with deeper coverage of both obvious and concealed security risks.

## Features
- AI-powered code security analysis using LLMs
- Automatic detection of common code vulnerabilities and hidden auxiliary vulnerabilities
- Lightweight and easy to integrate
- Inspired by Claude Code Security
- Supports multiple programming languages
